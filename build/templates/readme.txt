Purpose
=======

The Support Package contains validated configuration files that can be used as reference for Instantiation and Configuration.

Scope
=====

To support the deployment of "Dual-Mode 5G Core $TARGET_VERSION on Ericsson $TRACK", the delivery provides a support package containing the validated configuration example files.

See description for main directory as below:

* common: containing a CCD external network hot template
*  site1: containing geo-red site1 CCD and CNF configuration example files
*  site2: containing geo-red site2 CCD and CNF configuration example files

Version
=======
For accurate product versions, please refer to "Dual-Mode 5G Core $TARGET_VERSION on Ericsson $TRACK" Release Notes.
